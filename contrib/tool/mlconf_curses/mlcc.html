mlterm configurator on curses

What is mlterm configurator on curses?

A tiny tool to change terminal settings of mltermi.

If you have installed gtk, mlconfig can be used and this tool is not needed at all.
(mlconfig is a GUI configurator and should be more user-friendly)

Or you can also try "w3mmlconfig" if you have w3m.

If you are not using both of them, this tool may help you to change

  • Terminal encoding
  • X Input Method
  • Font size/color
  • Anti Aliasing
  • Wall picture
  • Transparent background
  • ... and more .

# Actually, you can send an escape sequence to your mlterm by hand and control its setting. For example, 

 echo -e "\e]5379;wall_picture=/path/to/file.jpg\x07"

will change wall picture to /path/to/file.jpg. This tool was written to automagically generate such a magic word instead of you.

Requirement

  • perl 5 (5.6 is recommended)
  • ncurses 5.2 or compatible (BSD curses ver4.2 is known to be problematic)

Install

It's a plain perl script. You do not have to do anything to run.

You can try by

perl mlconf_curses.pl

To install,

(If your perl is not in /usr/bin, edit "PERL=" line in Makefile.) 

make&make install

After installation, you should be able to run this tool by 

mlconf_curses

(.pl is omitted because it's ugly and someday this tool would be rewritten using one of other languages.)
